## Palette

Our design system build upon MUI palette and introduces new colors and hues. Palette can be customized via `palette`
prop in `ThemeProvider`:

```tsx
// Default palette
import { palettes } from '@elonkit/react';

<ThemeProvider palette={palettes}>...</ThemeProvider>;
```

## Typography

Typography is based on the guidelines from the Material Design spec and is arranged into typography levels. Each level
has a font-size, line-height, font-weight and letter-spacing. Typography can be customized via `createTypography` prop
in `ThemeProvider`:

```tsx
// Default typography
import { createTypography } from '@elonkit/react';

<ThemeProvider createTypography={createTypography}>...</ThemeProvider>;
```
