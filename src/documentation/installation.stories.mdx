# Installation

Elonkit React is available as an [npm package](https://www.npmjs.com/package/@elonkit/react). To install and save
dependencies in your `package.json`:

```bash
npm install @elonkit/react @mui/material @emotion/react @emotion/styled
```

Then follow the installation instructions for [MUI](https://mui.com/getting-started/installation/).
