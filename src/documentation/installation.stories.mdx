# Installation

Elonkit React is available as an [npm package](https://www.npmjs.com/package/@elonkit/react). To install and save
dependencies in your `package.json`:

```bash
yarn add @elonkit/react @mui/material@next @emotion/react @emotion/styled
```

Then follow the installation instructions for [Material-UI](https://next.material-ui.com/getting-started/installation/).
